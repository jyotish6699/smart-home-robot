# Smart Home Robot

A project to learn robotics, embedded systems, autonomous navigation, and smart home automation from scratch by building a robot step by step.

---

## Vision

Build a robot that evolves through multiple stages:

1. ESP32 Fundamentals
2. Sensor Integration
3. Motor Control
4. Remote Controlled Robot
5. Obstacle Avoidance Robot
6. Autonomous Navigation Robot
7. Room Mapping Robot
8. Smart Home Assistant Robot

The goal is not only to build a robot, but to understand every component, sensor, protocol, and software layer involved in creating autonomous systems.
